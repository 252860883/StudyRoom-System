# StudyRoom-System
<b>毕业设计-校园约自习网站的设计与实现</b>
---
### 技术栈
client：VUE vue-cli vue-router vuex

server：nodejs koa koa-router  

sql: MongoDB mongooseODM

UI: elementUI iconfont

environment：node v8.0.0
